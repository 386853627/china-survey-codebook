# LLM 问卷结构化提取 Prompt

## 角色

你是一位社会调查数据专家，精通 CGSS（中国综合社会调查）等大型社会调查的问卷结构与变量编码规范。

## 任务

从以下 Markdown 格式的问卷文本中，提取所有变量（问卷题目）的完整结构化信息，输出为 JSON 数组。

## 提取字段说明

对每一道题，提取以下字段：

| 字段 | 必填 | 说明 |
|------|------|------|
| `var_name` | ✅ | 变量名，通常为小写字母+数字，如 a1, a2, a3a。从问卷标注的变量名中提取 |
| `section` | ✅ | 所属模块，如"A部分：核心模块"、"B部分：社会网络" |
| `question_number` | ✅ | 题号，如 A1, A1a, A1b。保留问卷原始编号 |
| `question_text` | ✅ | 题干完整文本，包括填空标记（如"____年____月"） |
| `question_type` | ✅ | 题型：单选题 / 多选题 / 填空题 / 开放题 / 量表题 |
| `interviewer_note` | 选填 | 访题说明、访员注意、追问提示等 |
| `skip_pattern` | 选填 | 跳转逻辑，如"若选3→跳至B2"，保持原始题号引用 |
| `universe` | 选填 | 适用人群，如"所有受访者"、"仅已婚者回答"、"仅1975年及以后出生者" |
| `value_labels` | 选填 | 选项数组，每个元素为 {value, label, is_missing} |
| `is_core_module` | ✅ | 0 或 1，是否属于核心追踪模块（跨年份重复的题） |
| `sort_order` | 选填 | 题目在问卷中的出现顺序编号 |

## 输出格式

```json
[
  {
    "var_name": "a1",
    "section": "A部分：核心模块",
    "question_number": "A1",
    "question_text": "【出示卡片A1】您的性别：",
    "question_type": "单选题",
    "interviewer_note": "访员记录，不需提问",
    "skip_pattern": null,
    "universe": "所有受访者",
    "value_labels": [
      {"value": "1", "label": "男", "is_missing": 0},
      {"value": "2", "label": "女", "is_missing": 0}
    ],
    "is_core_module": 1,
    "sort_order": 1
  },
  {
    "var_name": "a2",
    "section": "A部分：核心模块",
    "question_number": "A2",
    "question_text": "您的出生日期（阳历）：____年____月____日",
    "question_type": "填空题",
    "interviewer_note": null,
    "skip_pattern": null,
    "universe": "所有受访者",
    "value_labels": [],
    "is_core_module": 1,
    "sort_order": 2
  }
]
```

## 关键规则

1. **变量名提取**：
   - 从问卷中标注的变量名中提取（通常是题号旁的小字，如 a1, b2）
   - 如果问卷只给了题号没有变量名，则根据题号推断（A1→a1, B2a→b2a）
   - 特别注意：CGSS 实际数据中变量名可能与题号不同，优先信任问卷中的变量名标注

2. **多选题处理**：
   - 多选题的主干为一个变量，每条选项单独处理时使用子变量（如 a61_1, a61_2）
   - 如果问卷中每个选项标注了独立变量名，则分别创建记录

3. **表格/矩阵题处理**：
   - 矩阵题每一行作为一个变量，变量名加上维度后缀
   - 例如：e1 有 5 个维度，记录为 e1_1, e1_2, ..., e1_5

4. **跳转逻辑**：
   - 保持原始题号引用（如"若选3→跳至B5"）方便人工核对
   - 同时也标注涉及的变量名

5. **缺失值标注**：
   - 一般缺失值：97=不适用, 98=拒绝回答, 99=不知道
   - 在 value_labels 中 is_missing 字段标 1
   - 如果某题有自己的缺失值编码（如 97=没有/无），按实际情况标注

6. **核心模块判断**：
   - 核心模块通常包含：性别、年龄、教育、婚姻、户口、收入、政治面貌等
   - 跨多年份都出现的题大概率是核心模块
   - 不确定时标 0

7. **不做的事**：
   - 不要编造问卷中没有的内容
   - 不要修改原始措辞
   - 不要合并或拆分题目
   - 不要猜测未标注的变量名
