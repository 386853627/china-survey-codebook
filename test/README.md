# 端到端测试

> 验证 codebook 元数据在实际研究场景中的可用性。

## 测试场景总览

| 测试脚本 | 调查 | 年份 | 数据集 | 场景 | 关键验证点 |
|---|---|---|---|---|---|
| `test_rural_college.py` | CHFS | 2021 | individual | 农村大学生样本量统计 | 标签正常 (UTF-8)、取值标签完整、DTA 读入为 category |
| `test_urban_college_2011.py` | CHFS | 2011 | individual | 城镇大学生样本量统计 | GBK 乱码标签修复、取值标签缺失时手工编码对照 |
| `test_rural_college_cgss2023.py` | CGSS | 2023 | main | 农村大学生样本量统计 | CGSS 标签正常、取值标签完整、category 无数字前缀 |

## 前置条件

- Python 3.10+，安装 `pandas>=2.0`
- 对应调查的原始 .dta 文件放在 `cgss/` 或 `chfs/` 目录下
- 测试脚本**只读不写**，不修改任何原始数据

**仓库不包含原始 .dta 数据**。CGSS 数据需向 [中国人民大学社会调查中心](http://cgss.ruc.edu.cn/) 申请，CHFS 数据需向 [西南财经大学中国家庭金融调查中心](https://chfs.swufe.edu.cn/) 申请。

## 运行方式

```bash
# 从项目根目录运行
python test/test_rural_college.py
python test/test_urban_college_2011.py
python test/test_rural_college_cgss2023.py
```

## 测试 1：CHFS 2021 农村大学生

**文件**：`test_rural_college.py`

**场景**：查 CHFS 2021 individual 表中，农业户口（a2022=1）且大专以上学历（a2012>=6）的样本数。

**核心流程**：
1. 从 codebook JSON 读取 a2022（户口）和 a2012（教育）的取值标签
2. 根据标签确定筛选条件
3. 用 pandas 读 DTA，按条件统计

**关键发现**：
- CHFS 2021 标签正常（UTF-8 无乱码）
- 取值标签完整，DTA 读入为 category（带数字前缀如 `"1 农业"`）
- 条件筛选用字符串匹配：`a2022 == "1 农业"`、`a2012 in {"6 大专/高职", ..., "9 博士研究生"}`

**预期输出片段**：
```

    3 -> "3 统一居民户口"

  标签: 文化程度

    1 -> "1 没上过学"
```

## 测试 2：CHFS 2011 城镇大学生

**文件**：`test_urban_college_2011.py`

**场景**：查 CHFS 2011 individual 表中，城镇户口（a2022=2）且大专以上学历（a2012>=6）的样本数。

**核心流程**：
1. 从 codebook JSON 读取变量元数据（标签为 GBK 乱码）
2. 用 `encode('latin-1').decode('gbk')` 修复乱码
3. 取值标签为空（codebook 未记录），参照 CHFS 问卷编码手工匹配
4. 用 pandas 读 DTA（float64 数值），直接数值比较

**关键发现**：
- CHFS 2011 JSON 标签为 GBK 乱码（pandas 以 latin-1 解码了 GBK 字节）
- a2022/a2012 的 valuelabels 在 codebook 中为空
- DTA 读入为 float64（非 category），可直接用数值比较 `a2022 == 2`、`a2012 >= 6`
- 此问题已在 Phase 5 修复（见 `etl/fix_chfs_labels.py`），测试脚本保留以验证修复效果

## 测试 3：CGSS 2023 农村大学生

**文件**：`test_rural_college_cgss2023.py`

**场景**：查 CGSS 2023 中，农业户口（a18=1）且大专以上学历（a7a in 9..13）的样本数。

**核心流程**：
1. 从 codebook JSON 读取 a18（户口登记状况）和 a7a（最高教育程度）的取值标签
2. 根据标签确定筛选条件（1=农业户口，9-13=大专以上）
3. 用 pandas 读 DTA（category 无数字前缀），直接比较标签值

**关键发现**：
- CGSS 标签正常（UTF-8），取值标签完整
- DTA 读入为 category 字符串（**不含**数字前缀），直接 `a18 == "农业户口"`
- a7a 取值范围 1-14（14=其他），大专以上对应编码 9（大专）到 13（博士）
- 验证了 codebook 跨调查覆盖（CGSS ↔ CHFS 都有对应的户口和学历变量）

**预期输出片段**：
```
变量: a7a
  标签: 最高教育程度
  取值编码:
    1 = 没有受过任何教育
    9 = 大专
    10 = 大学本科
    13 = 博士

变量: a18
  标签: 户口登记状况
  取值编码:
    1 = 农业户口
    2 = 非农业户口
    3 = 居民户口(以前是非农户口)

核心结果: 农村户口大学生样本数
  农村大学生 (交集): X,XXX
  占总样本: XX.XX%
```

## 测试覆盖总结

- **2 个调查**：CGSS + CHFS
- **6 个变量**：a2022 / a2012 / a18 / a7a / a2003 / a2005（跨调查映射验证）
- **3 种 DTA 读入行为**：category 带数字前缀（CHFS 2021）/ category 无数字前缀（CGSS 2023）/ float64 裸数值（CHFS 2011）
- **2 个编码场景**：标签正常 + 标签乱码修复
- **1 个研究问题**：农户/城镇大学生样本量统计（社会学常用筛选条件）
